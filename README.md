# Detecting Sponge Attack Against Multi-Exit Neural Networks

## 📌 Overview
This project focuses on detecting sponge attacks on Multi-Exit Neural Networks (MENs), based on the research paper:
"Sponge Attack Against Multi-Exit Networks With Data Poisoning".

The goal is to analyze how attackers increase inference latency without affecting classification accuracy and propose detection mechanisms.

---

## 🎯 Objectives
- Understand sponge attack behavior in MENs
- Analyze Exit Rate (ER) and latency changes
- Simulate attack conditions
- Design a detection approach

---

## 🧠 Key Concepts
- Multi-Exit Neural Networks (MEN)
- Sponge Attack
- Data Poisoning
- Exit Rate (ER)
- Entropy-based delay
